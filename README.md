# Nova Market

An optimized, multi-page ecommerce catalog built purely with native platform web solutions.

## Architecture Metrics
- **Zero JS Form Boilerplate**: Relies entirely on native browser HTML5 `required`, `minlength`, and RegEx `pattern` matching engine logic.
- **Combined Script Engine**: Unified API fetching routines, custom routing conditional checking, and LocalStorage updates managed inside one core script asset file (`app.js`).
- **Unified Layouts**: Clean media query layouts combined directly within a single semantic stylesheet framework (`style.css`).

live deployments: https://dedotre.github.io/Front-End-Development-Finals-project/
